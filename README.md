# FretLab 🎸

FretLab — это профессиональная гитарная веб-платформа и интерактивная DAW-подобная рабочая станция (Digital Audio Workstation), объединяющая в себе инструменты для ИИ-генерации соло-партий, детального анализа звука, занятий под бэк-треки со стримингов и глубокого изучения музыкальной теории прямо на грифе.

Проект разработан как отзывчивое одностраничное приложение (SPA) на базе React и TypeScript с упором на производительность звукового движка в реальном времени.

---

## 🚀 Ключевые возможности платформы

### 1. AI Studio & Музыкальный BBS-Движок
* **Синхронизированная генерация:** Умный генератор аккордовых прогрессий и мелодических соло-линий, жестко завязанный на тактовую сетку.
* **Сетка в стиле DAW:** Визуализация треков (Chords, Solo, Lead) на кастомной адаптивной SVG-сетке со штилями нот, отображающая длительности в реальном времени.
* **BBS-Тайминг (Bars:Beats:Sixteenths):** Вся логика воспроизведения `Tone.Part` переведена с абсолютных секунд на музыкальные доли. Это полностью исключает рассинхрон, микрозадержки и обрывы лупов при изменении BPM.
* **Автоматический бэнд и бас:** Движок автоматически генерирует плотную басовую линию (корневые ноты в нижнем регистре с пульсацией) и подстраивает соло под выбранный лад.

### 2. Студийный Микшер и Звуковое Ядро (`AudioManager`)
* **Гибридный аудиодвижок:** Платформа использует передовые возможности `Tone.js` и низкоуровневого `Web Audio API`.
* **Поддержка реальных сэмплов:** Ленивая загрузка живых аудиосэмплов через `Tone.Sampler` для гитары, баса и ударных установлена через относительные пути (идеально для деплоя на GitHub Pages).
* **Умный Fallback:** Если аудиосэмплы еще загружаются или отсутствуют на сервере, ядро автоматически переключает каналы на кастомные синтезаторы гармоник (`Tone.PolySynth` / `Oscillator`), сохраняя работоспособность студии.
* **Аппаратный микшер:** Встроенная в UI панель управления громкостью и мьютом (Mute/Unmute) для каждого канала в отдельности (Master, Guitar, Bass, Drums, Chords).

### 3. Интеграция бэк-треков (Streaming & Jamming)
* **Мультиплатформенный плеер:** Глобальный контекст поддерживает бесшовное подключение и управление бэк-треками (Backing Tracks) с крупнейших стриминговых сервисов: **YouTube, RuTube и VK (ВКонтакте)**.
* **Синхронизация стейта:** Возможность комфортно репетировать, зацикливать треки и отрабатывать гаммы поверх реального видео или оригинального минуса.

### 4. Визуализация теории и Определение нот на грифе
* **Интерактивная карта грифа:** Динамическое отображение ступеней и нот на виртуальном грифе гитары в строгом соответствии с выбранной тональностью.
* **Огромная база ладов:** Полная поддержка диатоники, пентатоники, блюзовой гаммы, гармонического/мелодического минора, симметричных ладов (Altered) и арпеджио (`maj7_arp`, `min7_arp`, `dom9_arp`).
* **DAW-автоскролл:** При воспроизведении трека виртуальный гриф и табулатура автоматически плавно скроллируются по вертикали вслед за играющей нотой.

### 5. Высокоточный цифровой тюнер
* **Алгоритм MPM (McLeod Pitch Method):** Вместо стандартного и неточного метода автокорреляции (ACF), тюнер FretLab использует математический алгоритм MPM.
* **Ноль задержки:** Точное определение гитарных частот и распознавание нот с микрофона в реальном времени с фильтрацией лишних гармоник.

---

## 🛠 Технологический стек

* **Frontend:** React 18, TypeScript, Сборщик Vite
* **Аудио-ядро:** Tone.js (Продвинутый транспорт, тайминги, сэмплеры и синтезаторы)
* **Низкоуровневый звук:** Web Audio API (Кастомный расчет частот, фаз и сложение гармоник струн)
* **Графика и UI:** Интерактивный кастомный SVG-рендеринг временных шкал без тяжелых внешних UI-библиотек.
